# 📊 MIS Bank Reporting — Advanced Excel

Multi-sheet MIS reporting workbook for UK bank data.

## Features
- SUMIFS, Goal Seek & Subtotals for balance analysis
- Advanced Filters (balance > £75,000)
- Conditional Formatting & Pivot/Donut Charts
- Macro Recording for automated reporting
- Audit-ready & print-ready output

## Tech Stack
- Microsoft Excel (Advanced)
- SUMIFS, Goal Seek, Subtotals, Pivot Charts, Macros
